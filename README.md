# ruijie
pwsh自动连接校园网，比python总要方便点吧

## 设置
在auto_connect里设置好以下三项：
```
userId
password
service
```
## 开机自启
- win+r, shell:startup

![image](https://user-images.githubusercontent.com/45586871/135025367-2369d1a8-6fab-4467-bfdc-1ac80c3b034e.png)
